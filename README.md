# House Price Prediction

A simple ML project to predict house prices based on features like area, bedrooms, bathrooms etc.

I built this using Linear Regression on a housing dataset from Kaggle.

## What it does

Takes house features as input and predicts the price of the house.

## Tech used

- Python
- Scikit-learn (Linear Regression, StandardScaler)
- Pandas, NumPy

## How to run

pip install pandas scikit-learn numpy
python house_price_prediction.py

Make sure housing.csv is in the same folder.

## Dataset

Housing Price Prediction dataset from Kaggle.

## Result

Used R2 Score to evaluate — higher the score, better the model.
